# Tangent — Branching AI Conversations
## Overview

Tangent is a new way to interact with AI — built for thinking, not just chatting.

Instead of linear conversations, Tangent lets you branch ideas, explore alternatives, and visualize your thinking.

Think: ChatGPT, but with threads like Discord + version control for ideas.

## ❌ The Problem

AI chat interfaces are limited:

- Conversations are linear
- Exploring multiple ideas = messy copy/paste
- You lose context when trying alternatives
- No way to compare different directions side-by-side

AI is powerful — but the interface is holding it back.

## 💡 The Solution

Tangent introduces branching conversations:

- Fork any message into a new direction
- Explore multiple ideas in parallel
- View branches side-by-side
- Navigate your thinking like a tree
``` Main Thread:
1 → 2 → 3 → 4

Branch from 3:
        ↘ 3 → 5 → 6
```
## 🧩 Key Features
### 🌿 Branching Conversations
- Create alternate paths from any message
- Never lose a thought again

### 🧠 Parallel Thinking
- Compare multiple AI responses at once
- Explore “what if” scenarios instantly

### 🧭 Visual Navigation
- Tree-based conversation structure
- Clean UI for complex ideas

### 💬 Context Preservation
- Each branch maintains its own context
- No more restarting conversations
